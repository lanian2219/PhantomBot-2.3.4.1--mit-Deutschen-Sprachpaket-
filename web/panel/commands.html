Befehle

(sender) Benutzer der den Befehl auslöste.
(@sender) Benutzer der den Befehl ausgelöst hat und es wird ein @ Zeichen seinem Namen hinzugefügt.
(baresender) Benutzer der den Befehl auslöste, in Kleinbuchstaben.
(touser) Ersetzt mit dem Argument, welches nach dem Befehl gesagt wird. Wenn danach nichts angegeben wird, dann ist es der Benutzer der den Befehl auslöst.
(#) Zufällige Nummer
(1), (2), (3) etc Befehlsargumente
(age) Zeigt an, wie lange du schon auf Twitch bist.
(random) Zufällige Person im Chat.
(pointname) Aktueller Punktename.
(uptime) Aktuelle Streamlaufzeit.
(game) Aktuelles Spiel auf dem Kanal.
(status) Aktueller Kanaltitel.
(price) Aktueller Befehlspreis. (Zuschauer & Stammzuschauer)
(pay) Aktuelle Kosten eines Befehls. (Mods)
(onlineonly) Befehl ist nur ausführbar, wenn Stream online.
(offlineonly) Befehl ist nur ausführbar, wenn Stream offline.
(viewers) Aktuelle Zuschauerzahl.
(count) Zeit, die der Befehl bereits verwendet wurde.
(echo) Sagt jede Nachricht als Bot.
(1=) Argument 1. Füge ein Argument nach dem = ein. Für wen, wenn das Argument leer ist.
(countdown=) Füge das Datum ein, zu dem du herunterzählen willst. Beispiel: (countdown=Juli 2 2016 7:00).
(downtime) Zeit seit dem letzten Stream.
(readfile filename.txt) Liest, was in der Datei steht. Die Datei muss im addons Ordner des Bots liegen.
(customapijson URL key | {literal string} ...) Rufe eine Remote-URL auf, die JSON zurückgibt. key oder object.key oder object.object.key für Zugriff auf JSON-Daten.
(command [command_name]) Befehl aufrufen. Muss das erste Element im benutzerdefinierten Befehl sein. Inkludiere kein ! vor command_name!
(customapi [requesturl]) Eine Remote-URL aufrufen.
